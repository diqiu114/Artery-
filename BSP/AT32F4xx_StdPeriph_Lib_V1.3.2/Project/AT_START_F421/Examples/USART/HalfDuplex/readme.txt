@par Example Description 

  This demo provides a basic communication between USART1 and USART2 in 
  Half-Duplex mode using flags.If received data right ,the Three leds will 
  turn on.
  
  Set-up 
  - Connect USART1 Tx pin (PA.02) to USART2 Tx pin (PA.09) and connect a 
  pull-up resistor to this line (10K).
